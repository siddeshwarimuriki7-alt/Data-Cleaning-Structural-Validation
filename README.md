## Objective
Clean raw data by removing duplicates, handling missing values, validating fields, and standardizing formats.

## Tools
- Python
- Pandas

## Steps
1. Load dataset
2. Remove duplicates
3. Handle missing values
4. Standardize date formats
5. Validate emails
6. Export cleaned dataset
